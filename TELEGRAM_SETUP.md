# Настройка Telegram алертов — 15 минут

## Шаг 1 — Создай Telegram бота (3 минуты)

1. Открой Telegram, найди **@BotFather**
2. Отправь `/newbot`
3. Придумай имя: `Sniper Pullback`
4. Придумай юзернейм: `sniper_pullback_bot` (должен заканчиваться на `bot`)
5. BotFather пришлёт **токен** — выглядит так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
6. Скопируй токен

## Шаг 2 — Получи свой Chat ID (2 минуты)

1. Напиши своему новому боту любое сообщение (например `/start`)
2. Открой в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
   (замени `<ВАШ_ТОКЕН>` на токен из шага 1)
3. В ответе найди `"chat":{"id":XXXXXXXXX}` — это и есть твой Chat ID
4. Запиши число (например `123456789`)

## Шаг 3 — Вставь в конфиг (1 минута)

Открой файл `telegram_config.py` и замени:
```python
TELEGRAM_TOKEN   = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
TELEGRAM_CHAT_ID = "123456789"
```

## Шаг 4 — Проверь что работает (1 минута)

```bash
cd d:\projects\trading
python scanner_alert.py
```

Должно прийти сообщение в Telegram.

---

## Шаг 5 — Настрой автозапуск (Windows Task Scheduler)

1. Открой **Планировщик задач** (Task Scheduler) — найди через поиск Windows
2. Нажми **"Создать простую задачу"** (Create Basic Task)
3. Заполни:
   - Имя: `Sniper Pullback Scanner`
   - Триггер: **Еженедельно** → Пн, Вт, Ср, Чт, Пт → время **22:05**
   - Действие: **Запустить программу**
   - Программа: `d:\projects\trading\run_scanner.bat`

4. Поставь галочку **"Открыть окно свойств"** перед финишем
5. В свойствах → вкладка **"Условия"** → убери галочку "Только при питании от сети"

Готово! Каждый будний день в 22:05 придёт Telegram сообщение с сигналами.

## Журнал сделок (journal.csv)

Все сигналы автоматически сохраняются в `journal.csv`.

Колонки:
- `date_scanned` — когда нашли
- `symbol`, `type`, `sector` — что нашли
- `entry`, `stop`, `target`, `rr` — уровни
- `result`, `result_date`, `pnl_pct` — **заполняешь сам** когда сделка закрылась
- `notes` — заметки

Раз в месяц спрашиваешь Клода: *"проанализируй journal.csv"* — получаешь разбор.
